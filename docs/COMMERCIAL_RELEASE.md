## Executive Summary

AegisGraph Sentinel Enterprise is now commercially available as a production-ready enterprise SaaS platform. This document outlines the product editions, pricing, go-to-market strategy, and customer onboarding process.

---

## Product Editions

### Community Edition (Free)

**Target Audience**: Individual researchers, students, open-source contributors

**Features**:
- Real-time fraud detection API
- Basic HTGNN model
- Up to 10,000 API calls/month
- 5 team members
- 10GB storage
- Community support
- Basic reporting dashboard

**Limitations**:
- No advanced AI agents
- No enterprise integrations
- No SLA guarantee
- Public documentation only

---

### Professional Edition ($499/month)

**Target Audience**: Small to medium businesses, growing fraud teams

**Features**:
- Everything in Community
- Advanced HTGNN Pro model
- Unlimited API calls
- 25 team members
- 100GB storage
- Email support (24h response)
- Custom dashboards
- Higher rate limits
- SSO integration (Google, Microsoft)
- Basic audit logging
- 3 enterprise integrations (choose from: Splunk, ServiceNow, Slack)

**Additional**:
- Dedicated onboarding session
- Quarterly business review

---

### Enterprise Edition ($1,999/month)

**Target Audience**: Large enterprises, banks, financial institutions

**Features**:
- Everything in Professional
- Custom HTGNN model training
- All AI agents (Investigation, Threat, Compliance, Forensics, Reporting)
- Unlimited team members
- Unlimited storage
- Priority support with 4h SLA
- Advanced analytics & BI integration
- All SSO providers (Okta, Azure AD, SAML)
- Full audit logging & compliance
- All enterprise integrations
- Dedicated account manager
- SOC2 Type II compliance
- Custom contracts & SLA
- On-premise deployment option
